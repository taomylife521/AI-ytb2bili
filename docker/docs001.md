### 快速使用指南
1. 部署后端服务
   ① 克隆/下载仓库：https://github.com/difyz9/ytb2bili-docker
   ② 进入仓库目录，编辑 `config.yml` 文件，将其中的 DeepSeek API Key 替换为你自己的有效密钥
   ③ 执行命令启动服务：`docker compose up -d`（确保本地已安装 Docker 且 Docker 服务正常运行）

2. 安装浏览器插件
   ① 下载最新版 Chrome 插件：https://github.com/difyz9/ytb2bili_extension
   ② 按照 Chrome 扩展程序安装指引（开发者模式）加载该插件

3. 功能使用
   安装完成后，打开 YouTube 视频播放页面，点击播放器上的专属功能按钮，即可一键完成 YouTube 视频及字幕的下载与翻译。
